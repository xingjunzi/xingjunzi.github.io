5、配置nginx和php服务

借助文档查询部署方案，第一步为系统安装编译工具链。

说明：提前安装编译所需工具与依赖库。build-essential 包含 gcc、make 等编译工具；Nginx 依赖 pcre、zlib、OpenSSL；额外库用于完善 PHP 扩展功能。

5.1 为系统装上编译工具链

# 更新软件包列表
sudo apt update

# 安装基础编译工具和常用库
sudo apt install -y build-essential libpcre3 libpcre3-dev zlib1g-dev openssl libssl-dev

# 扩展依赖库（增强 Nginx/PHP 功能）
sudo apt install -y libxml2-dev libxslt1-dev libgd-dev libgeoip-dev libbz2-dev \
libjpeg-dev libpng-dev libfreetype6-dev libzip-dev libonig-dev
基础编译工具安装终端输出
PHP/ Nginx 扩展依赖库安装终端输出

5.2 编译与安装 Nginx

5.2.1 下载和解压源码

从 Nginx 官网获取稳定版源码包，解压后进入目录编译。

5.2.2 创建系统运行用户

# 创建无法登录终端的 Nginx 运行用户 nginx
sudo useradd -s /sbin/nologin nginx
创建无登录权限系统用户操作截图

5.2.3 配置、编译与安装

Nginx ./configure 编译参数配置

5.2.4 设置 Nginx 系统服务

创建 systemd 服务文件：/etc/systemd/system/nginx.service

[Unit]
Description=The NGINX HTTP and reverse proxy server
After=network.target

[Service]
Type=forking
PIDFile=/var/run/nginx.pid
ExecStartPre=/usr/local/nginx/sbin/nginx -t
ExecStart=/usr/local/nginx/sbin/nginx
ExecReload=/usr/local/nginx/sbin/nginx -s reload
ExecStop=/bin/kill -s QUIT $MAINPID
PrivateTmp=true

[Install]
WantedBy=multi-user.target
Nginx 系统服务配置文件编辑界面

重载服务、开机自启并启动 Nginx

sudo systemctl daemon-reload
sudo systemctl enable nginx
sudo systemctl start nginx

6.2 配置与编译

PHP 编译 configure 参数配置界面

6.3 配置文件设置

PHP 运行配置文件编辑界面

6.4 设置 PHP-FPM 系统服务

创建 /etc/systemd/system/php-fpm.service 服务文件

[Unit]
Description=The PHP FastCGI Process Manager
After=network.target

[Service]
Type=simple
ExecStart=/usr/local/php/sbin/php-fpm --daemonize --fpm-config /usr/local/php/etc/php-fpm.conf
ExecReload=/bin/kill -USR2 $MAINPID
PrivateTmp=true

[Install]
WantedBy=multi-user.target

重载服务、启动并查看状态

sudo systemctl daemon-reload
sudo systemctl start php-fpm

7、配置 Nginx 与 PHP 协作

7.1 修改 Nginx 主配置文件

编辑配置文件：sudo nano /usr/local/nginx/conf/nginx.conf

server {
    listen 80;
    server_name localhost;
    root /usr/local/nginx/html;
    index index.php index.html index.htm;

    location ~ \.php$ {
        fastcgi_pass 127.0.0.1:9000;
        include fastcgi.conf;
    }

    error_page 500 502 503 504 /50x.html;
    location = /50x.html {
        root html;
    }
}
Nginx 支持PHP解析的站点配置
